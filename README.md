# Frontend Submodule

Frontend foundation for the browser-based multiplayer game (Milestone 1).

## Tech Stack

*   **Engine:** Vite
*   **Framework:** React + TypeScript
*   **Package Manager:** Bun
*   **Styling:** Tailwind CSS

---

## Local Development

You must have [Bun](https://bun.sh/) installed. Do not use `npm` or `yarn` to prevent lockfile conflicts.

1.  **Install dependencies:**
    ```bash
    bun install
    ```

2.  **Start the development server:**
    ```bash
    bun run dev
    ```

---

## Architecture Notes

*   All dependency management must be done natively via Bun (`bun add <package>`).
*   Legacy npm artifacts (`node_modules` generated by npm, `package-lock.json`) are explicitly banned from version control.
